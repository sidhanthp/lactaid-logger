# Lactaid Logger

Track your dairy intake, log Lactaid dosage, and get personalized recommendations.

## Features

- **Log Meals** — Search from 40+ common foods with pre-estimated dairy/lactose content, or enter custom foods
- **Track Lactaid Dosage** — Record how many Lactaid pills you took with each meal
- **Symptom Tracking** — Log symptoms (none/mild/moderate/severe) after meals
- **Personal Dosage Guide** — Machine-learned recommendations based on your history
- **Insights & Stats** — Success rate, trends, and tips

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- localStorage persistence
- Mobile-first, PWA-ready

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
